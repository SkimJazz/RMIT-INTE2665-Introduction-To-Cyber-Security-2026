# 31. Basic Kali Linux Security Practices

Suggested steps:

* Configure firewall inbound/outbound rules
* Change default SSH port (22)
* Enforce account lockout policies
* Apply strong password policies
* Regularly update system packages

---

## 📌 Notes

* These exercises build **Linux literacy**, not hacking
* Commands learned here are reused throughout the course
* Accuracy of understanding is more important than speed
